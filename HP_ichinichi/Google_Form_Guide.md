# Google フォーム 作成ガイド
## いち日写真館｜ご予約・お問い合わせフォーム

---

## ① フォームを新規作成する

1. [Google フォーム](https://forms.google.com) を開く
2. 「＋」（空白のフォーム）をクリック
3. 右上の**パレットアイコン**でカラーを選択（お好みで）

---

## ② タイトルと説明文をコピペ

**フォームのタイトル：**
```
ご予約・お問い合わせ｜いち日写真館
```

**説明文（タイトル下のテキスト欄）：**
```
ご予約・お問い合わせはこちらのフォームよりお気軽にどうぞ。

【大切なご注意】
・このフォームは予約確定ではありません。
　内容を確認のうえ、折り返しメールにてご連絡いたします。
　メールへの返信をもって予約確定となります。
・返信は通常2〜3日以内を予定しています。
　お急ぎの場合は LINE からご連絡いただくとスムーズです。
　▶ LINE: https://lin.ee/pSxLqqB
```

---

## ③ 質問を追加する（全6項目）

以下の順番で「質問を追加」ボタン（＋アイコン）を押して追加してください。

---

### 質問 1：お問い合わせの種類

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `お問い合わせの種類` |
| 形式 | **ラジオボタン（選択肢を1つ選ぶ）** |
| 必須 | ✅ ON |

**選択肢（1行ずつコピペ）：**
```
撮影のご予約
撮影のお問い合わせ
その他
```

---

### 質問 2：お名前

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `お名前` |
| 形式 | **記述式（短文回答）** |
| 必須 | ✅ ON |

---

### 質問 3：メールアドレス

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `メールアドレス` |
| 形式 | **記述式（短文回答）** |
| 必須 | ✅ ON |

**説明文（質問文のすぐ下に追加できます）：**
```
docomo・au・SoftBankなどキャリアメールの場合、こちらからのメールが届かないことがあります。
できればGmailなどのアドレスをご記入ください。
```

---

### 質問 4：電話番号

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `電話番号` |
| 形式 | **記述式（短文回答）** |
| 必須 | 任意（OFF のまま可） |

---

### 質問 5：撮影内容・ご希望

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `撮影内容・ご希望` |
| 形式 | **段落（長文回答）** |
| 必須 | ✅ ON |

**説明文：**
```
撮影ジャンル（家族写真・七五三・マタニティなど）、ご希望の撮影場所、お子様の年齢など、
わかる範囲でお知らせください。
```

---

### 質問 6：ご希望の撮影日

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `ご希望の撮影日` |
| 形式 | **記述式（短文回答）** |
| 必須 | 任意（OFF のまま可） |

**説明文：**
```
第1希望・第2希望など複数日あればすべてお書きください。
まだ決まっていない場合はその旨コメント欄にご記入ください。
```

---

### 質問 7：コメント・その他ご質問

| 設定項目 | 内容 |
|---------|------|
| 質問文 | `コメント・その他ご質問` |
| 形式 | **段落（長文回答）** |
| 必須 | 任意（OFF のまま可） |

## ④ 送信後メッセージの設定

1. 右上の**歯車アイコン**（設定）をクリック
2. 「プレゼンテーション」タブを選択
3. 「確認メッセージ」にコピペ：

```
お問い合わせいただきありがとうございます。
2〜3日以内にメールにてご連絡いたします。

お急ぎの場合は LINE からご連絡ください。
https://lin.ee/pSxLqqB

いち日写真館｜松田恵美
```

---

## ⑤ 回答をメールで通知する設定

1. フォーム編集画面で「回答」タブをクリック
2. 右上の**3点メニュー（︙）**をクリック
3. 「新しい回答についてのメール通知を受け取る」をクリック

→ フォームに送信があるたびに `cocoacrd@gmail.com` にメールが届きます。

---

## ⑥ フォームのURLをコピーしてサイトに貼る

1. 右上の「送信」ボタンをクリック
2. リンクアイコン（🔗）をクリック
3. 「URLを短縮する」にチェックを入れてコピー
4. contact.html に貼る場合はお知らせください。ボタンリンクを更新します。
